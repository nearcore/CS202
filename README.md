# CS202 Java Exam Review

Static review website for the Pokemon and FoodMenu Java practice projects.

It focuses on:

- abstract parent classes
- inheritance with `extends`
- constructor chaining with `super(...)` and `this(...)`
- `@Override`
- polymorphism with `List<Pokemon>` and `List<Food>`
- Swing `JPanel`, `paintComponent`, and mouse events

Open `index.html` in a browser to use the review site.
