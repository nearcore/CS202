Same Java Ideas, Two Projects

Study the pattern once, then recognize it in both Pokemon and FoodMenu.

Parent Class

Pokemon and Food are abstract parent classes. They store shared data and shared methods.

Child Classes

Charmander extends Pokemon. Adobo extends Food. Child classes customize the parent.

Constructors

Child constructors use super(...) to pass setup values to the parent constructor.

Polymorphism

List<Pokemon> stores many Pokemon types. List<Food> stores many food types.

Swing Drawing

Both apps use JPanel, paintComponent, Graphics, and repaint().

Mouse Events

Pokemon uses click and drag. FoodMenu uses click selection. Both read mouse x/y values.

Last-Minute Cram Sheet

Be ready to explain these in simple words.

abstract

A parent class or method that is incomplete by design. Child classes finish the missing parts.

extends

Creates inheritance. The child class receives fields and methods from the parent class.

super(...)

Calls the parent constructor. It must be first inside a child constructor.

this(...)

Calls another constructor in the same class. Used in Pichu and Bulbasaur.

@Override

Compiler check that a child method correctly replaces a parent method.

List<Parent>

Polymorphism. A parent-type list can hold many different child objects.

paintComponent

Swing calls this when the panel needs to redraw. Always call super.paintComponent(g).

MouseListener

An interface that requires mouse methods like mouseClicked or mousePressed.
